# Custos

Custos is a lightweight browser extension aimed at reducing tab chaos while studying or coding by organizing open tabs in real time.

The goal is to make focus effortless without forcing users to manually manage tabs.

## How to Load the Extension
1. Clone or download this repository  
2. Open your browser and go to:
   - Chrome / Brave: `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the `custos` folder

The extension will now be available in your toolbar.
